Principais Operações

Iniciar Processo

A gestão por processo é a base do SEI. Para criar qualquer documento, você deve iniciar um novo processo ou inserir em processo já existente.

Passos:
  • Menu Principal → INICIAR PROCESSO
  • Escolher o tipo adequado
  • Preencher dados de cadastro
  • Clicar em SALVAR
Use o campo filtro para encontrar tipos de processo rapidamente!

Gerenciar Documentos

Inclua, edite e assine documentos internos ou externos no sistema.

Incluir

Clique no ícone e escolha o tipo de documento

Editar

Selecione o documento e clique no ícone

Assinar

Na v3.0 é possível assinar direto no editor!

Tramitar Processos

Envie processos para outras unidades e controle prazos de retorno.

Como enviar:
  1. Selecione o processo na tela CONTROLE DE PROCESSOS
  2. Clique no ícone
  3. Informe a unidade de destino
  4. Configure retorno programado se necessário
Dica: Use o retorno programado para controlar prazos. Processos atrasados aparecem em vermelho!

Sistema de Pesquisa

A v3.0 trouxe melhorias significativas na pesquisa, incluindo busca dentro dos documentos.

Tipos de Pesquisa:
  • Livre: Campo na barra de ferramentas
  • Estruturada: Menu → PESQUISA
  • No Processo: Nova funcionalidade v3.0
Conectores Lógicos:
  • E: termo1 E termo2
  • OU: termo1 OU termo2
  • NÃO: termo1 NÃO termo2

Sugestões

Melhores Práticas

Evite ao máximo a impressão de documentos eletrônicos. Use o SEI como ferramenta principal de gestão documental.

Otimização de Fluxo

Use marcadores na v3.0 para organizar processos por categoria e prioridade, facilitando a visualização.

Segurança

Sua senha é pessoal e intransferível. Configure lembretes para trocá-la periodicamente.
